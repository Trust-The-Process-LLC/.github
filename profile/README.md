## 🎮 Current Project — *System Breach*

**Play as a rogue android navigating a high-security facility, where every shadow is an ally and every step could be your last.**

### Overview

*System Breach* is a DOOM-inspired stealth-action shooter that blends tactical infiltration with high-intensity combat.

### Core Gameplay

* Hack doors, security cameras, and alarm systems
* Use stealth to outmaneuver enemies and control the environment
* Transition seamlessly from silent movement to aggressive combat
* Navigate a hostile facility while uncovering a deeper narrative

### Tech Stack

* **Engine:** Unity 6
* **Platforms:** Windows, Linux, macOS *(Console planned)*
* **Release Window:** Q4 2026
